# Ractime

Time Tracking App
- track daily/weekly/monthly time spent for different things
- Main Landing page shows gantt chart for the day and a pie chart of the week
- there is a page where Categories/Tags for tasks can be created
    - they recieve a name and a color
    - Categories would be something like: University
- Diagrams are shown by category, so there would be a part of the pie chart for each category
- On the Main landing page a task can be stared
    - the tasks recieves a name, a description and a category/tag
    - the time starts running for this task and is beeing recorded until you click on stop again


## Technical Stuff
- Purely Browser Based Web App
    - all data is stored in the browser storage
- Vue.js for the page
- make it look like a racoon theme wise
